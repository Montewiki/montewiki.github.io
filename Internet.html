Fonctionnement d'Internet

Explorez le fonctionnement d'Internet et découvrez les protocoles, outils et technologies qui permettent la communication mondiale.

Internet a ses origines dans le projet ARPANET, créé dans les années 1960 par le département de la Défense des États-Unis. Son objectif était de créer un réseau de communication décentralisé permettant de partager des informations efficacement entre différentes universités et centres de recherche.

Exemple : Si vous envoyez une lettre par le biais du courrier traditionnel, cette lettre passe par plusieurs bureaux jusqu'à atteindre son destinataire. Sur Internet, les paquets d'informations suivent un parcours similaire à travers différents réseaux et appareils jusqu'à atteindre leur destination.

Le **protocole TCP (Transmission Control Protocol)** est l'un des piliers de la communication sur Internet. Il garantit que les données envoyées d'un point à un autre arrivent complètes et sans erreur.

Exemple : Imaginez que vous envoyez une lettre avec plusieurs pages. Si une page se perd en chemin, le protocole TCP vous permet de la renvoyer sans avoir à renvoyer toute la lettre.

Exemple de paquets TCP :
Paquet 1 : "Bonjour"
Paquet 2 : "Monde"
Paquet 3 : "!"

Le **protocole IP (Internet Protocol)** est responsable de diriger les paquets de données vers leur destination. Chaque appareil sur Internet possède une **adresse IP unique**.

Exemple : Si vous avez une adresse chez vous (par exemple, Rue Fictive 123), cette adresse est unique et permet aux facteurs de savoir où livrer votre courrier.

Exemple d'adresse IP :
IPv4 : 192.168.1.1
IPv6 : 2001:0db8:85a3:0000:0000:8a2e:0370:7334

Le **DNS (Domain Name System)** permet de traduire les **noms de domaine** en adresses IP. C'est l'annuaire téléphonique d'Internet.

Exemple : Lorsque vous tapez "www.exemple.com" dans votre navigateur, le DNS effectue la recherche pour traduire ce nom en l'adresse IP correspondante, comme "192.168.1.1".

Exemple de requête DNS :
Requête DNS : www.exemple.com → 192.168.1.1
